# Trapelpokah. ✈️

Landing page website travel yang dibuat dengan melakukan **slicing design dari Figma** ke dalam kode HTML & CSS.

## 🎨 Design Reference

Design ini berasal dari **Figma Community**:

🔗 [Travling - Travel Website Landing Page (Community)](https://www.figma.com/design/RiSnszioIgA8ttPUrFt1pR/Travling----Travel-Website-Landing-Page--Community-?node-id=0-1&t=1eud1hv3b7UxZ5Cf-1)

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Flexbox, CSS Grid, Media Queries
- **Google Fonts** — Poppins
- **Font Awesome** — Icons

## 📱 Responsive

Website ini sudah responsive di berbagai ukuran layar:

| Breakpoint | Device           |
| ---------- | ---------------- |
| `> 1024px` | Desktop          |
| `≤ 1024px` | Tablet Landscape |
| `≤ 768px`  | Tablet Portrait  |
| `≤ 480px`  | Mobile           |
| `≤ 360px`  | Small Mobile     |
